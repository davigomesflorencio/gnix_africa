# Africa (gnix_africa)

Africa is a beautiful and interactive iOS application built with SwiftUI that serves as a comprehensive encyclopedia for African animals. Users can explore various species through detailed galleries, interactive maps, insightful videos, and educational facts.

## üöÄ Features

- **Animal Profiles**: Detailed information about various African animals, including their habitats, diets, and conservation status.
- **Interactive Map**: A MapKit-integrated view showing the locations and habitats of different species across the African continent.
- **Video Gallery**: A collection of high-quality videos showcasing animals in their natural environments.
- **Dynamic Image Gallery**: Explore stunning photography with a flexible grid layout system.
- **Educational Facts**: Interactive and engaging facts about each animal to enhance learning.
- **iMessage Stickers**: A dedicated sticker pack extension included for use in the Messages app.

## üõ† Technologies Used

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Mapping**: MapKit
- **Video Playback**: AVFoundation
- **Persistence/Data**: JSON-based data management
- **Extensions**: iMessage Sticker Pack Extension

## üì± Requirements

- **iOS**: 15.0+
- **Xcode**: 13.0+

## üìÅ Project Structure

- `gnix_africa/View/`: SwiftUI views for the main screens and reusable components.
- `gnix_africa/App/`: Main application lifecycle and primary entry points.
- `gnix_africa/Model/`: Data models for animals, videos, and locations.
- `gnix_africa/Data/`: JSON data files containing the application's content.
- `gnix_africa/Helper/`: Utility functions and extensions.
- `Stickers/`: Assets and configuration for the iMessage sticker pack.

## üõ† Installation

1.  Clone the repository.
2.  Open `gnix_africa.xcodeproj` in Xcode.
3.  Select a simulator or physical device running iOS 15.0 or later.
4.  Build and run the project (`Cmd + R`).
